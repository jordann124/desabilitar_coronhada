# desabilitar_coronhada

Simplesmente só desativa o R 

# Instalação
1. Instala O Script
3. Copie o projeto para a tua pasta de recursos do esx.
4. Não se esqueça de importar o arquivo database.sql para o teu banco de dados.
5. Adicione "start desabilitar_coronhada" no teu arquivo `server.cfg`
